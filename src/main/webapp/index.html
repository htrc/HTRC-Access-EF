HT + HTRC Mashup

Overview

Installing the software provided through this web page allows you to mashup HTRC's Bookworm and Extracted Features data-set with the main HathiTrust web site. Once the software is installed when you next visit www.hathitrust.org you will notice a few additional features:

  • Bookworm Button. Next to the regular search button there will now also be a button with a worm icon on it. Having entered some terms into the search box, you now have a choice: pressing the magnifying search button to initiate a regular search within the HathiTrust DL, or else pressing this new button next to it to initiate a bookworm search with the terms you have just entered.
  • Extracted Features Integration. Along side links for Catalog Record and Full view for volumes/documents in the interface, there will additional be links for download Extracted Features. You find these links appearing in search result sets, and when browsing the items that make up publicly available collections. While most HT volumes have an extracted feature counterpart in HTRC, due to the on-going expansion of the content of the HathiTrust, not all of them do. In such circumstances the Mashup software marks these entries as HTRC Unfriendly.
  • HT Collection to HTRC Workset.In the collections area of the HathiTrust it is possible to download metadata for the currently selected collection. Now an additional download button is available—Collection to Workset—that downloads the same metadata, but filtered to those that are available as extracted features in the HTRC.

Installation

To try out this HT + HTRC Mashup you need to have user-scripting enabled in your browser. This is a two-step process:

  1. Install a user-script manager extension/add-in to your web browser such as GreaseMonkey (Firefox) or TamperMonkey (other browsers: Chrome, Microsoft Edge, Safari, Opera etc.). The former is the add-in, developed for Firefox, that became the catalyst for the more general idea of user-scripting in browsers. The latter is a project that has put emphasis on being cross-browser compatible.
  2. Add the HT+HTRC Mashup Script to your user-scripting extension/add-in:
    • Mashup Version 1.2

Now visit www.hathitrust.org and look out for the additional buttons and links!
